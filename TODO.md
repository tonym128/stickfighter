# StickFighter Release Status

## Core Completed Features
- [x] High-performance Bone/Skeletal Engine with 60fps smoothing.
- [x] 10 Unique playable characters with different stats and faces.
- [x] Universal Fireball Special Move (↓ → + A).
- [x] Circular Input Buffer (60 frames) with diagonal leniency.
- [x] Arcade Ladder Mode with scaling AI difficulty.
- [x] Dynamic Camera system with automatic scaling and floor-tracking.
- [x] Multi-platform support: Arduboy (hardware) and SDL2 (Desktop).
- [x] Multi-window desktop Animation Editor for development.
- [x] Polished main menu with Logo and animated UI.
