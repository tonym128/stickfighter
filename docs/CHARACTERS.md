# StickFighter Roster

Each character uses the same base bone structure but has different `Bone Lengths`, `Speeds`, and `Move Sets` stored in PROGMEM.

## Character Archetypes
1.  **Zenith (All-Rounder):** Balanced speed and range. Standard fireball/uppercut.
2.  **Cinder (Speedster):** High speed, low health. Can dash through opponents.
3.  **Goliath (Grappler):** Slow, high health, massive bone lengths. Command grabs.
4.  **Volt (Zoner):** Long-range limb strikes and projectile focus.
5.  **Kage (Assassin):** Can teleport or turn invisible briefly.
6.  **Siren (Counter):** Large parry window, high damage on successful parry.
7.  **Drift (Floaty):** Low gravity, aerial combat specialist.
8.  **Tusk (Bruiser):** High chip damage on blocks. Hard-hitting kicks.
9.  **Jade (Rushdown):** Fast attack recovery, relies on long combos.
10. **Echo (Mimic):** Copies the move set of the opponent.

## Special Move Structure
Each character has 3 Specials + 1 Super.
- **Specials:** Triggered by input sequences (e.g., QCF + A).
- **Super:** Triggered by `A + B` with a full Special Bar.

### Example: Zenith
- **A+B (Super):** *Omni-Strike*. Zenith dashes across the screen, dealing multiple hits if it connects.
- **QCF + A:** *Plasma Bolt*. A fast projectile.
- **DP + A:** *Rising Knee*. Invincible startup, great anti-air.
- **QCB + B:** *Spin Kick*. Hits twice, good for combos.
